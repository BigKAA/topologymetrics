# Документация Go SDK

## Обзор

dephealth Go SDK обеспечивает мониторинг здоровья зависимостей для Go
микросервисов через метрики Prometheus. Поддерживает выборочный импорт
для минимального размера бинарника и реализацию пользовательских чекеров.

**Текущая версия:** 0.8.0 | **Go:** 1.22+
